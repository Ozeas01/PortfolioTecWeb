Análise de Servidor com Sobrecarga em Nuvem

O treinamento "Olimpíadas Azure 2024" capacitou profissionais em soluções cloud com foco na plataforma Microsoft Azure. Através de teoria e prática, os participantes resolveram um problema de sobrecarga em servidor, aplicando conhecimentos de infraestrutura, redes e cloud computing.

Nullbank - Sistema de Gestão Bancária

Um sistema completo para clientes, funcionários e gerentes com login, transações e controle de contas. Usou servidor local WampServer, banco de dados via phpMyAdmin e interface web para gerenciamento.

Tecnologias: WampServer, MySQL, PHP, HTML, CSS, JavaScript

AutoTech House - Automação Residencial

Projeto prático das disciplinas STR e IoT, com sensores e atuadores conectados via Wi-Fi/Bluetooth em uma casa inteligente em maquete. Permite controle remoto e automação de tarefas em tempo real.

Tecnologias: Arduino, IoT, STR, HTML, Bluetooth, Wi-Fi
