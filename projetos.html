Analise servidor com sobrecarga em nuvem

O treinamento "Olimpíadas Azure 2024" capacitou profissionais em soluções cloud com foco na plataforma Microsoft Azure. Com 30 horas de carga horária, o curso combinou teoria e prática por meio de um projeto real, que envolveu a resolução de um problema de sobrecarga em servidor, utilizando conhecimentos em redes, infraestrutura e computação em nuvem. A experiência preparou os participantes para oferecer soluções escaláveis, seguras e eficientes no mercado.

Tecnologias: Python, Sklearn, Streamlit, Pandas, Matplotlib e Seaborn.

Nullbank - Sistema de Gestão Bancária

Desenvolvimento de um sistema de gestão bancária com funcionalidades para clientes, funcionários e gerentes, integrando login, transações e gerenciamento de contas. O projeto incluiu a implementação de um servidor local usando WampServer, com configurações de banco de dados via phpMyAdmin e interface web para transações.

Tecnologias: WampServer, MySQL, PHP, HTML, CSS, JavaScript.

AutoTech House - Sistema de Automação Residencial

AutoTech House é um projeto desenvolvido nas disciplinas de Software em Tempo Real (STR) e Internet das Coisas (IoT). O projeto consiste em uma maquete de casa inteligente equipada com sensores e atuadores que permitem a automação e o monitoramento remoto via web e dispositivos móveis, ele integra o controle de tarefas em tempo real com conectividade Wi-Fi e Bluetooth. Com o código disponível no GitHub, este projeto proporcionou um aprimoramento significativo das habilidades em IoT, automação e programação, aplicando na prática os conceitos aprendidos em STR.
